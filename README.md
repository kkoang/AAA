关于征订《2017年福建省高等职业教育入学考试考试大纲的说明》的通知

各市、县（区）教育招生考试机构、平潭综合实验区教育考试中心、各中学，各中等职业学校：								
为了帮助广大普通高中及中等职业学校的教师和学生及时了解2017年福建省高等职业教育入学考试的命题指导思想与命题原则、能力层次与考核目标、考试内容与考试要求以及考试形式与试卷结构等考试大纲的具体要求，福建省教育考试院组织专家编写了《2017年福建省高等职业教育入学考试考试大纲的说明》。该书既是2017年高等职业入学考试的命题依据，也是我省高等职业教育入学考试最具权威性、系统性、实用性、可读性的复习参考用书。该书由厦门大学出版社出版，将于2016年11月上旬出版发行。现将该书的征订工作事项通知如下：
一、征订资料
１、《2017年福建省高等职业教育入学考试考试大纲的说明（面向普通高中）（语文、数学、英语、信息技术）》
２、《2017年福建省高等职业教育入学考试考试大纲的说明（面向中职学校）（语文、数学、英语）》
３、《2017年福建省高等职业教育入学考试考试大纲的说明（面向中职学校）（专业基础知识）》
二、征订办法
1.请各设县(市、区)教育招生考试机构协助通知本辖区内各中学，学校根据订阅自愿的原则鼓励学生征订。
2.为确保考试资料及时送达，请各征订单位务必于2016年10月25日前将征订回执发至我部邮箱（2742986379@qq.com），收到订单后我部会以邮件形式确认订单。请各征订单位于10月25日前将款项汇至福州兴才考生书刊服务部账户，汇款单上需注明汇款单位及用途，如：福州三中   2017年高职大纲说明。
三、征订要求
1.款到才发货。征订单位应在指定时间前汇款，如在指定时间后征订款还未发出，我部视相关征订单位的《征订回执》无效。
2.款项须校对。征订单位汇款后请将“汇款时间、汇款用途、汇款单位名称、汇款金额、汇款联系人姓名和联系电话”等信息告知我部。我部业务经办人登记上述事项后，经与财务人员核对，如未收到款项或收到款项不符会打电话给征订单位的经办人确认相关事项。
3.学校经办人应尽可能提供QQ号，并及时加入我部中学服务工作群。QQ群名称：福建中学考生服务（群号：316062607）。本群负责征订数据核对、账务核对及物流配送业务，请参与征订的学校指定一名工作人员加入本群，并将用户名设为“学校+姓名”。
4.我部将核对QQ工作群中各学校经办人员信息，维护工作群环境，保障发行通畅。
5.多协商处理。征订单位如有修改联系人、联系电话和发货地址，及汇款校对出现误差，敬请主动联系我部，我部将妥善认真解决。
